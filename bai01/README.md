<html lang="vi">
<head>
    <meta charset="utf-8" />
    <title>CSS3 Nhập môn | La Văn Thanh</title>
    <style></style>
</head>
<body>
    <h1>CSS3 Nhập môn</h1>
    <p>Tôi tên là thanh rất vui được chia sẻ tới các bạn khóa học này</p>
</body>
</html>
~~~
