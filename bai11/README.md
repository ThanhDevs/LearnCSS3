Thẻ Danh sách
=============

## Nội dung chính

* Thẻ danh sách：ul, ol, li
* Kiểu ：list-style-*

### Website tham khảo

https://developer.mozilla.org/en-US/docs/Web/CSS/list-style

## Thực hành

### index.html

~~~html
<!DOCTYPE html>
<html lang="zh-cmn-Hans">
<head>
    <meta charset="utf-8" />
    <title>CSS3 nhập môn | La Văn Thanh</title>
    <link rel="stylesheet" type="text/css" href="style.css" />
</head>
<body>
    <h1>Danh sách</h1>
    <ul>
        <li>Tin tức</li>
        <li>Thể thao</li>
        <li>Khoa học</li>
    </ul>
    <ol>
        <li>Python</li>
        <li>Javascript</li>
        <li>Ruby on Rails</li>
    </ol>
</body>
</html>
~~~
